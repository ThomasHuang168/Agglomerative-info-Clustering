The source code requires Eigen C++ library, which is included in the subfolder Eigen and also available at
http://eigen.tuxfamily.org/

demo.cpp is an example that performs the exact and approximate clustering algorithm on an AWGN model.

install cmake and openCV first
create a build directory for executable file
select include folder as source path and newly created bin folder as binary path
select configure and then add the openCV source path and then configure again
if no error appear in configure, click generate

build the software with the selected conpiler
